## Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- Backend API running on `http://localhost:3000`

## Installation

1. Clone the repository
```sh
git clone <repository-url>
cd vue-ticker
```

2. Install dependencies
```sh
npm install
```

3. Run dev (only works with ticker-api)
```sh
npm run dev
```
